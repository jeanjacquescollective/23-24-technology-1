File formats

CSV

  • Comma Separated Values
  • tabulaire gegevens
  • platte tekst met komma's om waarden te scheiden
  • gebruikt in excel om gegevens te delen
                        Naam, Leeftijd, Stad
                        Alice, 28, New York
                        Bob, 32, Los Angeles
                        Carol, 24, Chicago

JSON

  • JavaScript Object Notation
  • Populaire voor gegevensuitwisseling
  • tekst met attribuut-waarde paren
                        {
                            "naam": "Alice",
                            "leeftijd": 28,
                            "stad": "New York"
                        }
                        {
                            "naam": "Bob",
                            "leeftijd": 32,
                            "stad": "Los Angeles"
                        }
                        {
                            "naam": "Carol",
                            "leeftijd": 24,
                            "stad": "Chicago"
                        }

XML

  • Extensible Markup Language
  • Veelzijdige taal voor mens- en machineleesbare documenten.
  • Gebruikt voor gegevensuitwisseling en configuratie.
                        
                            Alice
                            28
                            New York
                        
                        
                            Bob
                            32
                            Los Angeles
                        
                        
                            Carol
                            24
                            Chicago

YAML

  • YAML is not a Markup Language
  • leesbare gegevensindeling voor configuratie en uitwisseling
  • inspringing voor structuur
                        - naam: Alice
                          leeftijd: 28
                          stad: New York
                        - naam: Bob
                          leeftijd: 32
                          stad: Los Angeles
                        - naam: Carol
                          leeftijd: 24
                          stad: Chicago

Excel Werkblad

  • XLSX-bestanden voor tabulaire gegevens en analyse.
  • Ondersteunt werkbladen, formules en opmaak.

SQLite Database

  • Serverloze SQL-database voor ingebedde systemen en mobiele apps.

Protobuf

  • Protocol Buffers
  • binair
  • gebruikt in Google

MessagePack

  • Binaire serialisatie-indeling
  • goede prestaties en efficiëntie
  • gegevensuitwisseling tussen systemen in verschillende talen

Parquet

  • Kolomsgewijze indeling
  • gebruikt in Hadoop en andere big data-tools.
  • Goed voor snel lezen data

Avro

  • Gegevensserialisatieraamwerk
  • binaire indeling
  • Gebruikt in Hadoop en andere big data-tools.
  • Goed voor snel schrijven data

ORC

  • Optimal Record Columnar
  • open-source
  • kolomsgewijs
  • Goed voor snel lezen data

HDF5

  • Hierarchical Data Format
  • grote hoeveelheden wetenschappelijke gegevens
  • Ondersteunt complexe structuren en hiërarchieën
