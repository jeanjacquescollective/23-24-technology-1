Data Sources

Relational Databases

Relational databases are widely used for structured data. They offer ACID (Atomicity, Consistency, Isolation, Durability) properties and are ideal for transactional systems.

Common Relational Databases:

  • MySQL
  • PostgreSQL
  • Microsoft SQL Server
Example

NoSQL Databases

NoSQL databases are flexible and suitable for unstructured data. They include:

  • Document Stores (e.g., MongoDB)
  • Key-Value Stores (e.g., Redis)
  • Column-family Stores (e.g., Cassandra )
  • Graph Databases (e.g., Neo4j)

APIs

Application Programming Interfaces

APIs enable data access from online services.

  • RESTful APIs
  • GraphQL
  • SOAP APIs

Streaming Data Sources

Examples

  • Apache Kafka
  • Amazon Kinesis
  • Apache Flink

Streaming data sources provide real-time data. They are used for applications requiring immediate data processing and analysis.

Data Storage

Data Warehouses

Data warehouses are repositories for historical data used for reporting and analysis. Examples include Snowflake and Amazon Redshift.

Cloud Storage

Cloud storage services like Amazon S3 and Google Cloud Storage provide scalable, durable storage for various data types.

Data Lakes

Data lakes store raw, unstructured data that can be processed and analyzed later. Examples include AWS Glue and Azure Data Lake Storage.

Specialised Data sources

IoT Data Sources

IoT devices generate massive amounts of data. Collecting and analyzing this data is crucial for IoT applications.

Social Media Data

Social media platforms offer APIs for accessing user-generated content and engagement data.

Geographic Data

Geographic information systems (GIS) provide spatial data used in maps, navigation, and urban planning.

Big Data Sources

Big data sources include massive datasets generated by businesses, social media, and scientific research.

Data from External Partners

Collaborating with external partners often involves sharing and integrating data from different sources.
