Understanding Traditional Programming

Definition of Traditional Programming

  • Strict rules
  • Deterministic
  • Human-defined

Examples of Traditional Programming


import time

def wake_up():
    print("Alarm rings...")
    time.sleep(2)
    print("Hit snooze...")
    time.sleep(2)
    print("Hit snooze again...")
    time.sleep(2)
    print("Promise to wake up...")

    
if __name__ == "__main__":
    wake_up()

Introduction to Machine Learning

What is Machine Learning?

What is Machine Learning?

  • Learn from data
  • Make predictions
  • Discover patterns
  • Subset of AI

How Machine Learning Differs

  • Not explicitly programmed
  • Not deterministic
  • Not human-defined

Types of Machine Learning

  • Supervised
  • Unsupervised
  • Reinforcement

Supervised Learning

  • Learn from labeled data
  • Regression & Classification
  • External supervision
  • Maps the labeled inputs to the known outputs
  • More accurate

Unsupervised Learning

  • Learn from unlabeled data
  • Association, Clustering & Dimensionality Reduction (preprocessing)
  • No supervision
  • Understands patterns & discovers the output
  • Hidden patterns
  • No predictions

Semi-supervised learning

  • Learn from labeled and unlabeled data
  • Combines supervised and unsupervised learning
  • Uses unlabeled data to improve supervised learning
  • Uses supervised learning to label unlabeled data

Reinforcement Learning

  • Learn from feedback - reward-based
  • No predefined data
  • No supervision
  • Trial-and-error
  • Agent interacts with environment
